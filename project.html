Outline

The project must consist of between 3 and 8 charts, tables and visualisations.

The project must discuss four topics: 1. The aims of the project, 2. the data you used, how you accessed it including notes on automation/replication, 3. Challenges in data cleaning or analysis and the tools you used to overcome them, 4. Your conclusions.

The written text of the project must not exceed 800 words.

The criteria used to assess the project are given below.

Criteria

1. Accessibility

a. Hosted on a GitHub page - open data approach.

b. Clear links to data and notes in your code.

Mobile first: site should be clear on a smartphone screen.

2. Empirical Design

a. Clarity of question

b. Choice of data sets

c. Empirical approach

3. Automation

a. Use of APIs

b. Replicability of project

4. Impact

a. Key takeaways from the project

b. Interactivity of the data

5. Description

a. Clarity of write up

b. Logical steps from data to conclusions

c. Does the project stimulate further work?

Data Science Project

I scraped data from politico to get data on 2020 election outcomes for house seats.

I then scraped data from OpenSecrets to get financial data for house seats in the 2020 election.

How does campaign finance spending affect elections to the US House of Representatives?

Aims

We begin by scraping data from Politico and Open Secrets.

The code to scrape and merge these datasets and the final dataset can be found here.

We then plot graphs showing correlation between amount spent and vote share.

We then consider the total spent on each seat.

We create a dataset showing that which can be found here.

We then plot a map showing where the most was spent.

We see this seems to track with wealthy states.

We then use census data to consider whether household income correlates with amount spent on each seat.

We create this dataset using this code.

We then plot a map or correlation showing whether that correlates.

We then consider if in-state funding matters.

We use data from OpenSecrets and merge with this code to create this dataset.

We then plot a graph showing whether in-state funding matters, and compare to normal funding.
