How does campaign finance spending affect elections to the US House of Representatives?

Aims

I scraped data from Politico to get data on 2020 election outcomes. I then scraped data from OpenSecrets to get financial data for house seats in the 2020 election.

I then cleaned and merged these datasets to create this dataset.

We then plot graphs showing correlation between amount spent and vote share.

We then consider the total spent on each seat.

We use this code (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/code_snippets/house_2020/district_totals.py) to produce this dataset (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/data/house_2020/district_totals_2020.csv)

We then plot a map showing where the most was spent.

We see this seems to track with wealthy states.

We then use census data to consider whether household income correlates with amount spent on each seat.

So we get census data (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/data/house_2020/census_household_income_2019.csv) and merge it (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/code_snippets/house_2020/household_income_merge.py) to produce this dataset (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/data/house_2020/district_totals_household_income_2020.csv)

We then plot a map or correlation showing whether that correlates.

We then consider if in-state funding matters.

We use data from OpenSecrets (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/data/house_2020/house_2020_in_out_district_open_secrets.csv) and clean it (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/code_snippets/house_2020/in_out_district_merge.py) to produce this dataset (https://github.com/Jacob-McLaughlin/Jacob-McLaughlin.github.io/blob/main/project/data/house_2020/in_distr_raised_2020.csv)

We then plot a graph showing whether in-state funding matters, and compare to normal funding.
